Isearch or rather Instagram search is a tool that retrieves information on an instagram profile (only public to all)

**Features of script**
 - fully async
 - asynchrone scraping 
 - menu in cli format (commands)

**Features**
 - id
 - username
 - fullname
 - biography
 - profile picture
 - number of followers
 - number of following
 - externals links
 - public / private account
 - buisness account
 - verified account
 - public phone number (+lookup)
 - public email
 - related profile
 - potentiels emails
 - download of profile picture
